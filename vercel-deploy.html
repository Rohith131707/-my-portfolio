Deploy to Vercel

Now you can deploy your portfolio to Vercel with these steps:

  1. Go to https://vercel.com/new
  2. Sign in with GitHub (if not already signed in)
  3. Select the repository: Rohith131707/-my-portfolio
  4. In the configuration screen:
    • Framework Preset: Next.js
    • Root Directory: portfolio (important!)
    • Build Command: next build
    • Output Directory: .next
  5. Click "Deploy"
Important: Make sure to set the Root Directory to portfolio since your Next.js project is in a subdirectory.

After Deployment

Once your site is deployed:

  • Vercel will provide you with a URL (like my-portfolio-username.vercel.app)
  • Any future changes you push to your GitHub repository will automatically be deployed
  • You can add a custom domain in the Vercel dashboard if desired

Troubleshooting

If you still encounter issues:

  • Check the build logs in Vercel for specific errors
  • Make sure you selected portfolio as the Root Directory
  • Try deploying directly from the Vercel dashboard
